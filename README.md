# Setup

Pre-Requisites:

- Ubuntu system running 26.04 LTS

1. Clone this repo.
2. Run the following:

```bash
cd <repo>
sudo apt update
pip install --upgrade pip
pip install pre-commit
sudo apt install plantuml
pipx install cmake-format
 sudo cp /usr/share/plantuml/plantuml.jar /usr/bin/
```
